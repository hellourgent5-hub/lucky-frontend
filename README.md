
# Poncherla Full Project (Starter)

This archive contains three folders:
- poncherla-backend : Node.js + Express + MongoDB starter
- poncherla-frontend : React + Vite user app
- poncherla-admin : React + Vite admin panel

Instructions are in each package's package.json scripts and .env.example files. Fill MONGO_URI and JWT_SECRET before running backend.
